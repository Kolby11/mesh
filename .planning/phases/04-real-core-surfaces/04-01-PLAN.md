Make the audio command contract executable end to end before quick-settings depends on it.

This plan reconciles the command shape and locks regression coverage.
Output: Contract-compatible frontend proxy command tests and audio providers that accept normalized volume payloads while preserving legacy `percent` compatibility if needed.
</objective>
