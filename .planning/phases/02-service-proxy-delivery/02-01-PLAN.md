<threat_model>
| Threat | Severity | Mitigation |
|--------|----------|------------|
| Missing contract/provider failures are swallowed by `pcall(require, ...)` and the UI degrades silently | high | Emit a visible diagnostic alongside the Lua error path, including plugin id, requested interface, and version range. |
| Implicit `on_<service>_update()` handlers fire in addition to explicit subscriptions in a way that duplicates side effects unpredictably | medium | Define and test a fixed ordering: apply payload, update bound globals, invoke implicit handler once, then invoke explicit subscriptions once. |
| Proxy field reads expose stale state after `emit_unavailable()` or repeated updates | medium | Keep the full payload table refreshed on every service event and add repeated-update tests that assert the current values only. |
| Service command routing regresses while touching shared shell service wiring | low | Preserve `mesh.<service>.<command>` channel translation and keep existing command-publication tests green. |
</threat_model>
