# Plan 43-04 - Prototype Comparison and Phase 44 Handoff

<objective>
Compare the Blitz and MESH-owned focused-crate prototype evidence under the same headings, record the selected Phase 44 production-proof path, and preserve enough reproduction detail for the next phase without designing the full migration plan.
</objective>

<must_haves>
- D-01: Compare structural and behavioral parity, not pixel-perfect production fidelity.
- D-02: Accept structured output where screenshots exceeded scope.
- D-03: Cover hover, click, slider movement/release, and open-close behavior for both paths.
- D-04: Confirm both paths used the same scenario fixture.
- D-07: Treat Blitz blocker evidence as valid only when concrete and reproducible.
- D-09: Keep Skia/rust-skia as fallback evidence only.
- D-10: Use the mandated seven headings for both paths.

</must_haves>
