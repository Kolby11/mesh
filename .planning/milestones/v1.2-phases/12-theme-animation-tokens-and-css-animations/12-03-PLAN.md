# Plan 12-03 - Renderer Keyframe Playback Primitives

## Objective

Complete renderer-side keyframe playback primitives so validated percentage keyframes can produce an in-flight animatable style at any timestamp, with fill mode, direction, iteration count, easing, and pause semantics.

## Design Truths

- Keyframe playback is percentage-stop based; no `from`/`to` aliases.
- The renderer interpolates the same transition-safe visual property set used by shell transitions.
- Paused animations freeze current state; infinite animations never finish while still attached.
- Fill mode controls pre-delay and post-completion displayed style.

<threat_model>
T-12-03: Incorrect playback math could leave infinite animations stuck dirty forever, skip fill-mode boundaries, or diverge from transition interpolation.
Mitigation: centralize transition-safe style interpolation, test finite and infinite timelines, and verify paused/fill-mode behavior with deterministic timestamps.
</threat_model>
