# Plan 13-05 - Real-Surface Shell Proof and Responsive Test Coverage

## Objective

Finish Phase 13 by proving the migrated navigation bar on the real shell module with automated tests for controls, selection presence, motion continuity, and constrained-width behavior.

## Design Truths

- Tests should use the real navigation-bar module wherever practical.
- One constrained-width proof is required, not optional.
- Docs only supplement the proof; they do not replace it.

<threat_model>
T-13-05: The nav bar can look correct in implementation files while key behaviors silently regress, especially after richer layout changes and motion additions.
Mitigation: extend existing real-surface shell tests, add a constrained-width assertion path, and keep author docs aligned with the shipped proof so later edits do not drift from the runtime contract.
</threat_model>
