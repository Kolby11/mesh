| T-05-02: Semantic lookup reads outside configured pack roots | high | Canonicalize pack roots and resolved candidate paths where possible, and accept semantic candidates only when they remain under the configured root or the bundled Material root. |

| T-05-04: Executor hand-rolls incomplete XDG theme traversal | medium | Use the existing `icon` crate adapter in `xdg.rs` for configured root lookup; retain only minimal explicit-path compatibility in `lib.rs`. |
</threat_model>
