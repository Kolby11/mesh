<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| frontend script -> service proxy | Untrusted plugin Luau code attempts to call methods on a service proxy. |
| frontend script -> shell event router | Untrusted plugin Luau code can publish arbitrary event channel strings. |
| shell router -> backend service command | Routed commands cross into backend provider command handlers that may mutate host state. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04-01 | Tampering | `reactive_values_equal()` | mitigate | Compare nested object/array values by actual `serde_json::Value` equality so same-length service payload changes dirty UI state. |
| T-04-04-02 | Elevation of Privilege | `create_service_proxy()` | mitigate | Require `service.<name>.control` before exposing contract command methods; read capability remains state-only. |
| T-04-04-03 | Elevation of Privilege | `script_events_to_requests()` | mitigate | Carry source capabilities through `PublishedEvent` and reject forged `mesh.<service>.<command>` events without control capability. |
| T-04-04-04 | Elevation of Privilege | `Shell::dispatch_service_command()` | mitigate | Re-check control capability before sending `BackendServiceCommand` to provider handlers. |
| T-04-04-05 | Repudiation | denied service command routing | mitigate | Emit diagnostics or warning logs that name the denied service control capability. |
</threat_model>
